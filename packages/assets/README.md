# @local-monorepo/assets

Assets Layer - Media management and external API integrations for images and files.

## Overview

This layer handles all asset-related functionality including upload, download, and external API integrations for media content.

### Features

- **File Upload/Download**: Single and bulk image uploads
- **External API Integration**: Integration with image APIs like Pexels API
- **Media Management**: Complete media asset lifecycle management
- **Bulk Operations**: Support for bulk image processing and management

## Setup

Make sure to install the dependencies:

```bash
pnpm install
```

## Development

Start the development server:

```bash
pnpm dev
```

## Scripts

- `pnpm dev` - Start development server with playground
- `pnpm build` - Build the layer
- `pnpm lint` - Run ESLint
